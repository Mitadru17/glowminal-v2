# DesignSystem.md

> Defines the visual language for every Glowminal interface.

# Design Principles

-   Calm over flashy
-   Scientific over decorative
-   Premium through whitespace
-   Accessible by default
-   Explain before recommending

------------------------------------------------------------------------

# Color Tokens

## Primary

Primary: #3CA956 Primary Dark: #2F8143

## Surfaces

Background: #F5F7F4 Surface: #FFFFFF

## Text

Primary: #222722 Secondary: #5B6A5C

## Feedback

Success: #6FD18B Warning: #E8B84D Error: #D64545

Divider: #E2E8E1

Never introduce blue or neon accents into the core brand.

# Typography

Primary Font: Inter

Fallback: Geist

Scale

H1: 48 H2: 36 H3: 28 H4: 22 Body: 16 Small: 14 Caption: 12

Line height: 1.5 body 1.2 headings

# Spacing

Base unit: 8px

4 8 12 16 24 32 40 48 64 96

Use consistent spacing throughout.

------------------------------------------------------------------------

# Border Radius

Small: 8 Medium: 12 Large: 20 Full: 9999

------------------------------------------------------------------------

# Shadows

Use subtle elevation only.

Cards: Soft shadow.

Dialogs: Medium elevation.

Avoid dramatic shadows.

------------------------------------------------------------------------

# Grid

Desktop max width: 1280px

Content width: 1200px

8px spacing system.
